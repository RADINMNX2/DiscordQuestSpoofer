## üì• Installation

1.  Download the `DiscordQuestSpoofer.plugin.js` file.
2.  Open **Discord User Settings**.
3.  Scroll down to **BetterDiscord** (or Vencord) -> **Plugins**.
4.  Click **Open Plugins Folder**.
5.  Drag and drop the downloaded file into the folder.
6.  Enable the plugin in the list.
